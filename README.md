# Learning Playwright

A learning project for Playwright automation framework focused on test automation and web scraping.

## Overview

This repository contains practice scripts and examples for learning Playwright, a modern web automation library for testing and automation tasks.

## Project Structure

- `01_Basics.js` - Fundamental Playwright concepts and usage
- `02_sampleprogram.js` - Sample automation programs
- `.gitignore` - Git ignore rules for the project

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn

## Installation

```bash
npm install playwright
```

## Usage

Run any script using Node.js:

```bash
node 01_Basics.js
node 02_sampleprogram.js
```

## Learning Topics

- Browser automation basics
- Element selection and interaction
- Page navigation and handling
- Test automation fundamentals
- Web scraping techniques
